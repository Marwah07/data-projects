# Project Skripsi: Prediksi Kesehatan Mental Mahasiswa

Project ini menggunakan dataset prodi dan IPK mahasiswa untuk membangun model prediksi kesehatan mental dengan metode:

1. SVM + SMOTE
2. SVM + ADASYN
3. Naive Bayes + SMOTE
4. Naive Bayes + ADASYN

## Struktur Folder
- ADASYN_Naive_Bayes_PSO.ipynb
- ADASYN_SVM_PSO.ipynb
- SMOTE_Naive_Bayes_PSO.ipynb
- SMOTE_SVM_PSO.ipynb
- clean_data_ADASYN_without_outliers.csv
- clean_data_SMOTE_without_outliers.csv

- .gitignore

## Cara Menjalankan
1. Buka notebook di Google Colab atau Jupyter Notebook
2. Pastikan dataset.csv berada di folder yang sama
3. Jalankan cell notebook satu per satu

## Tools / Library
- Python 3.x
- pandas
- numpy
- matplotlib
- scikit-learn
- imbalanced-learn (SMOTE & ADASYN)
